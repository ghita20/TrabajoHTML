Estructura con etiquetas HTML5

Ahora vamos a ir dando forma a la parte visible de la página web(lo que el usuario final verá).
Para la siguiente estructura, utilizaremos las nuevas etiquetas de HTML5.
Anteriormente la estructura del <body> se definía con etiquetas <div> pero gracias a las nuvas actualizaciones de HTML ahora nos será más facil estructurar el contenido final.

<header>

En esta sección definiremos lo que contendrá nuestra cabecera. Dentro de ella podríamos poner por ejemplo un texto de introducción.

<nav>

Elementos de nuestro menú de navegación. Dentro de el incluiremos enlaces a las diferentes páginas de nuestra página web.

<aside>

Esta sección se utiliza para incluir contenido no relevante sobre nuestra página. Por ejemplo: publicidad.

<section> y <article>

La etiqueta <section> y <article> se utilizan para poder "partir" nuestro contenido en diferentes secciones.

Por ejemplo: si tenemos un articulo sobre cocina podemos crear un <article> con el nombre de la receta, una etiqueta <section> con los ingredientes, otro etiqueta <section> con "preparación"...etc
Una etiqueta <section> o <article> puede contener otras etiquetas section/article dentro. Se pueden utilizar las dos de manera indistinta.

<footer>

Sección que identifica al pie de página. En esta sección podemos poner por ejemplo la información del autor o contacto.

Todas las etiquetas anteriores necesitan ser cerradas.
